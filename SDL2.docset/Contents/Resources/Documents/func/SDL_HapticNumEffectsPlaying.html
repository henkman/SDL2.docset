SDL_HapticNumEffectsPlaying

Use this function to return the number of effects a haptic device can play at the same time.

Syntax

int SDL_HapticNumEffectsPlaying(SDL_Haptic* haptic)

Function Parameters

haptic

the SDL_Haptic device to query maximum playing effects

Return Value

Returns the number of effects the haptic device can play at the same time or a negative error code on failure; call SDL_GetError() for more information. Should there be a link to Remarks here?
